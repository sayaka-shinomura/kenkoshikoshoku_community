# 健康志向食コミュニティ

## サイト概要
### サイトテーマ
摂取したい栄養素を含む野菜をつかった料理のレシピを検索したり、自分が作った野菜料理のレシピを投稿できるコミュニティサイト


### テーマを選んだ理由
健康の為に野菜の摂取を心がけているのですが、料理を作ろうと思うと、レシピのレパートリーが少なく、悩んでいました。<br>
また、野菜に含まれている栄養素はそれぞれ異なる為、摂取しようとしている野菜が自分の体に必要とされているものであるか判断が難しいと感じていました。<br>
自分の体調から必要としている栄養素を特定し、摂取すべき野菜とその野菜をつかったレシピを知ることができるサービスがあれば便利だと考え、このテーマにしました。

### ターゲットユーザ
* 健康を意識した、野菜を取り入れた食生活を実践したい人
* 自分の体調に適した食材選びに困っている人

### 主な利用シーン
* 自分の体調に適した野菜や栄養素の知りたい時
* 作りたい野菜料理のレシピを検索する時
* 自分が作った野菜料理のレシピを投稿する時
